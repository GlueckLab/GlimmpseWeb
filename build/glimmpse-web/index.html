Welcome to GLIMMPSE

This software allows researchers to find appropriate power and sample size for studies that will use the general linear multivariate model for data analysis. Common examples of this model include:

  • One sample t-test
  • Paired t-test
  • Two sample t-test
  • Analysis of variance (ANOVA)
  • Analysis of covariance (ANCOVA)
  • Repeated measures analysis of variance
  • Multivariate analysis of variance (MANOVA)
  • Multivariate analysis of covariance (MANCOVA)
  • Linear regression

This project was developed by Dr. Sarah Kreidler at the University of Colorado, department of Biostatistics and Bioinformatics.

The power calculations are based on the work of Professor Keith Muller at the University of Florida. Appropriate citations for the supporting publications are available on the related publications page.

Acknowledgments

This power software has evolved through multiple versions, with a major revision roughly every decade, usually corresponding to a language change. GLIMMPSE extends an existing program called POWERLIB. Bercedis Peterson and Keith Muller wrote POWERLIB for SAS PROC MATRIX (Version 0) in 1983. In 1992, Lynette Keyes and Keith Muller converted POWERLIB to SAS/IML (Version 1). Keyes also created a new manual, based on the PROC MATRIX manual. Douglas Taylor created code for creating confidence limits for power values based on random estimates in 2001, which Keith Muller then used to enhance POWERLIB. In 2003, Jacqueline Johnson and Keith Muller created Version 2 of POWERLIB (Johnson et al. 2009). Additionally, James Slaughter greatly improved the _PROBF module, and Matthew Gurka conducted extensive tests. At the same time, the related manual was significantly revised from the previous version.

POWERLIB requires matrix format input, while GLIMMPSE provides a Graphic User Interface (point-and-click) format. The new software also supports matrix input. In addition, GLIMMPSE 1.0.1 extends POWERLIB to include the work of Glueck and Muller (2003).

We are grateful to USEPA for support and applications underlying version 0. Research support from NICHD similarly drove the development of version 1. NCI and other NIH institutes research supported the development of versions 2 and 3, both directly and indirectly.
